## はじめに

DigitalCloneVでは外部サービスとしてStyleBertVits2を利用しています。<br>
StyleBertVits2はGithubで公開されている音声合成AIです。<br>
[Style-Bert-VITS2](https://github.com/litagin02/Style-Bert-VITS2)<br>
ここではその導入方法を説明します。<br>
StyleBertVits2公式が用意してくださった解説動画もありますのでこちらも参考にしてください。<br>
[Style-Bert-VITS2解説動画](https://www.youtube.com/watch?v=aTUSzgDl1iY)<br>

## インストール

DigitalCloneVの中に「Install-Style-Bert-VITS2_edit_limit.bat」というファイルがあることを確認してください。<br>

![sbv2](/kiokuya/images/sbv2/sbv2_bat.png)

赤枠のファイルを任意の場所に移動してください。<br>
StyleBertVits2は15GBくらいあるので余裕のあるドライブを使用してください。<br>
また、フォルダの階層に日本語表記があると不具合を起こすことがあるので全ての階層で半角英数字になっている場所に移動するようにしてください。<br>
移動したらそのファイルをダブルクリックしてインストールしてください。<br>
インストールには少し時間がかかります。<br>

![sbv2 install cui](/kiokuya/images/sbv2/sbv2_install_cui.png)

![sbv2 install editor](/kiokuya/images/sbv2/sbv2_install_editor.png)

インストールが完了するとコマンドライン(黒い画像のもの)に起動ログが表示され、ブラウザにStyle-Bert-VITS2エディターが起動します。<br>
利用規約に同意して、コマンドラインとブラウザを×マークを押して閉じてください。<br>

![sbv2 folder](/kiokuya/images/sbv2/sbv2_folder.png)

batファイルを実行した場所にStyle-Bert-VITS2がダウンロードされていることを確認してください。<br>

![sbv2 folder init](/kiokuya/images/sbv2/sbv2_folder_init.png)

Style-Bert-VITS2フォルダの中の「Initialize.bat」をダブルクリックして実行してください。<br>

![sbv2 folder init bat](/kiokuya/images/sbv2/sbv2_folder_init_bat.png)

「Press any key to continue ...」と出ていたらEnterキーを押して閉じてください。<br>
これでひとまずStyleBertVits2のインストールが完了です。<br>
